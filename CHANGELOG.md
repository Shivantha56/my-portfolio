## [1.0.0] - 2026-06-01

### Added
- Initial Next.js 14 portfolio project setup with App Router
- Tailwind CSS integration for styling
- Custom Geist font (variable fonts: GeistVF and GeistMonoVF)
- Netlify deployment configuration via `netlify.toml` with Node.js 22 runtime
- GitHub Actions CI pipeline with lint validation and production build check on every push and pull request to `main`
- GitHub Actions release workflow that auto-publishes a GitHub Release from version tags and extracts matching changelog notes
- `CHANGELOG.md` following Keep a Changelog conventions
